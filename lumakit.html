Prismix Docs

A compact Roblox utility library for service lookup, safer Instances, path helpers, tweens, notifications, timing wrappers, and executor-specific features when they exist.

Single Source.lua Tools table Clean failure reasons GitHub Pages docs

Load the current public build and keep the helper table short.

local Prismix = loadstring(game:HttpGet("https://raw.githubusercontent.com/SairyTheKing/prismix/main/Source.lua"))()
local t = Prismix.Tools

Create instances safely. Parent is assigned after all other properties.

local part, reason = t.create("Part", {
  Name = "PrismixPart",
  Anchored = true,
  Parent = workspace
})

if not part then
  warn(reason)
end

Tween a property with readable settings.

t.tween(part, {
  Transparency = 0.45
}, {
  time = 0.35,
  easingStyle = "Quad",
  easingDirection = "Out"
})

Wrap noisy callbacks with debounce or throttle.

local clickOnce = t.debounce(0.5, function()
  print("clicked")
end)

Install

Use the raw GitHub URL for the current published source.

local Prismix = loadstring(game:HttpGet("https://raw.githubusercontent.com/SairyTheKing/prismix/main/Source.lua"))()
local t = Prismix.Tools
`Tools`, `Helpers`, `Functs`, and `Functions` point to the same table. Use `Tools` for new scripts.

Toolbox

Character player, character, humanoid, rootPart, teleportTo

Examples

Find A Remote

local remote, reason = t.waitForPath(game.ReplicatedStorage, "Remotes.Quest", 10)
if not remote then
  warn(reason)
end

Extend Prismix

Prismix.Register("tpSpawn", function()
  return t.teleportTo(workspace.SpawnLocation)
end)

Prismix.Alias("spawn", "tpSpawn")
t.spawn()

Compatibility

`AstraKitDebug`, `LumaKitDebug`, `AstraKitBanner`, and `LumaKitBanner` still work as fallback setting names. Legacy helper aliases such as `pfireclickd` and `pfireproximitypromt` are also kept.

Use `Prismix.Tools` in new scripts. `Functs` exists only to avoid breaking older scripts.
