LumaKit Docs

Small Roblox helper library with simple names and clean failure messages.

Version 1.4.1 Single file Lua 5.1 syntax checkable

Install

local LumaKit = loadstring(game:HttpGet("https://raw.githubusercontent.com/SairyTheKing/lumakit/main/Source.lua"))()
local f = LumaKit.Functs
Use the SairyTheKing/lumakit raw URL above for the current public source.

Settings

Set options before loading the library.

getgenv().LumaKitDebug = true
getgenv().LumaKitBanner = true

Core Helpers

Helper Use
say(...) Print values.
warnmsg(...) Warn values.
err(message, level) Throw an error.
service(name) Return a Roblox service or nil, reason.
create(className, properties) Create an Instance. Parent is assigned last.

Create Example

local part, reason = f.create("Part", {
  Name = "LumaPart",
  Anchored = true,
  Parent = workspace
})

if not part then
  warn(reason)
end

Interaction Helpers

Helper Return
fireClickDetector(detector) true or false, reason
fireTouchInterest(source, target) true or false, reason
fireProximityPrompt(prompt) true or false, reason 
local ok, reason = f.fireProximityPrompt(workspace.Door.ProximityPrompt)
if not ok then
  warn(reason)
end

Extend

Add project helpers without editing Source.lua.

LumaKit.Register("tp", function(part)
  game.Players.LocalPlayer.Character:PivotTo(part.CFrame)
end)

f.tp(workspace.SpawnLocation)

Optional Helpers

Optional functions register only when the current environment supports them.

if f.available("writefile") then
  f.writefile("log.txt", "ready")
end

Validate

Run the local smoke test from the LumaKit folder.

lua validate.lua
