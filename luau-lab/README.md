# LuauLab

Beginner Roblox Luau tutorials that teach by making small things in Roblox Studio.

## What It Has

- 15 practical lessons
- Step by step object setup
- Copyable Luau code
- Explanation for what the code does
- Common mistakes
- Test checklists
- Quick reference for beginner concepts

## Lessons

- Click Door
- Talk GUI
- Teleport Pad
- Screen Button
- Damage Brick
- Coin Pickup
- Light Switch
- Speed Pad
- Checkpoint
- Tool Giver
- Shop Button
- Moving Platform
- Color Clicker
- Round Timer
- Server Message

## Use

Open `index.html` in a browser.

No build tools are required. The site is one static HTML file.
