Common Fixes

Nothing happens

  • Check the script type. UI needs LocalScript.
  • Check object names in Explorer.
  • Open Output and read the red error.

Infinite yield warning

Your code is waiting for an object name that does not exist. Fix the name or move the object to the right place.

Touched runs too much

Add a debounce table or cooldown value. Touch events can fire many times from one player.

Works in Studio, not online

Do important rewards, tools, coins, and purchases on the server. Use RemoteEvents when UI needs to ask the server.
