AstraKit Docs

A small Roblox helper library for common utility calls, safer instance creation, and executor-specific helpers when the environment supports them.

Single Source.lua Clean nil/false reasons Extensible helpers

Load the current public build and keep the helper table short.

local AstraKit = loadstring(game:HttpGet("https://raw.githubusercontent.com/SairyTheKing/astrakit/main/Source.lua"))()
local f = AstraKit.Functs

Create instances with properties. Parent is applied last.

local part = f.create("Part", {
  Name = "AstraPart",
  Anchored = true,
  Parent = workspace
})

Interaction helpers return a boolean and a reason when blocked.

local ok, reason = f.fireProximityPrompt(workspace.Door.ProximityPrompt)
if not ok then
  warn(reason)
end

Install

Use the raw GitHub URL for the current published source.

local AstraKit = loadstring(game:HttpGet("https://raw.githubusercontent.com/SairyTheKing/astrakit/main/Source.lua"))()
local f = AstraKit.Functs
The returned table is named by your variable. `AstraKit.Functs`, `AstraKit.Helpers`, and `AstraKit.Functions` point to the same helper table.

Settings

Set options before loading the library.

getgenv().AstraKitDebug = true
getgenv().AstraKitBanner = true
Older `LumaKitDebug` and `LumaKitBanner` names still work as compatibility fallbacks.

Helpers

Core Print, warn, error, try, service, create.
Character player, character, rootPart, waitFor.
Optional Executor/file helpers register only when available.
Helper Use
say(...) Print values.
warnmsg(...) Warn values.
err(message, level) Throw an error.
try(callback, ...) Run a function through pcall.
service(name) Return and cache a Roblox service, or nil, reason.
create(className, properties) Create an Instance. Parent is assigned last.
rootPart(source) Resolve a Player, character Model, BasePart, or local player to a BasePart.
waitFor(parent, name, timeout) Safe wrapper around WaitForChild.

Interactions

These helpers depend on executor support. They return true when they work, or false, reason when blocked.

Helper Expected input
fireClickDetector(detector) ClickDetector
fireTouchInterest(source, target) Player, character Model, BasePart, or nil local player plus target BasePart.
fireProximityPrompt(prompt) ProximityPrompt

Extend

Add project-specific helpers without editing Source.lua.

AstraKit.Register("tp", function(part)
  local root = f.rootPart()
  if root and part then
    root.CFrame = part.CFrame
  end
end)

f.tp(workspace.SpawnLocation)
