Install Prismix

Prismix is a simple Roblox Luau utility library for newer scripters who want cleaner scripts without learning a big framework first.

Version 4.1.0 Single Source.lua Beginner-friendly API
Load Prismix 
local Prismix = loadstring(game:HttpGet("https://raw.githubusercontent.com/SairyTheKing/prismix/main/Source.lua"))()
local t = Prismix.Tools

Use Prismix.Tools for new scripts. Helpers, Functs, and Functions are compatibility names for the same table.

What Prismix Is For

Prismix keeps common Roblox script work short: services, Instances, paths, attributes, character parts, events, cleanup, tweens, HTTP, remotes, notifications, and optional executor checks.

Cleaner Roblox Calls

Use t.service, t.create, t.set, and t.destroy instead of repeating setup.

Readable Lookup

Use t.get, t.waitForPath, t.ofClass, and t.tagged for object lookup.

Safer Flow

Use t.try, t.retry, t.waitUntil, t.debounce, and t.cleaner.

Small Utilities

Use t.clone, t.merge, t.clamp, t.round, and t.contains.

Prismix does not replace Luau. It makes beginner scripts shorter, easier to read, and easier to debug.

Return Pattern

Most helpers return the value you wanted. If a helper cannot run, it usually returns nil, reason or false, reason.

Check the result 
local value, reason = t.someHelper()

if not value then
  warn(reason)
  return
end

Quick Start

Create a part, check the result, then tween it.

Create and tween a part 
local Prismix = loadstring(game:HttpGet("https://raw.githubusercontent.com/SairyTheKing/prismix/main/Source.lua"))()
local t = Prismix.Tools

local part, reason = t.create("Part", {
  Name = "PrismixPart",
  Anchored = true,
  Size = Vector3.new(4, 1, 4),
  Position = Vector3.new(0, 6, 0),
  Parent = workspace
})

if not part then
  warn(reason)
  return
end

t.tween(part, {
  Color = Color3.fromRGB(64, 156, 255)
}, 0.35)

Services And Instances

Use service for cached Roblox services. Use create to build Instances and set Parent last.

Get Services
Services 
local Players = t.service("Players")
local ReplicatedStorage = t.service("ReplicatedStorage")
Create Instances
Instance 
local label = t.create("TextLabel", {
  Text = "Prismix",
  Size = UDim2.fromScale(1, 0.1),
  Parent = screenGui
})

Finding Things

Use path helpers when objects are nested. Use class, tag, and attribute helpers when the exact object name is not enough.

Wait for a remote path 
local ReplicatedStorage = t.service("ReplicatedStorage")

local remote, reason = t.waitForPath(ReplicatedStorage, "Remotes.Quest", 10)
if not remote then
  warn(reason)
  return
end
  • t.find(parent, name, recursive) safely finds a child.
  • t.get(parent, "A.B.C") finds a dot-separated path now.
  • t.waitForPath(parent, "A.B.C", timeout) waits through a dot-separated path.
  • t.ofClass(root, className, recursive) returns matching Instances by class.
  • t.tagged(tagName) returns CollectionService tagged Instances.

Player And Character

Character helpers accept simple sources: no argument for the local player, a Player, a character Model, a BasePart, a Vector3, or a CFrame.

Move to a target 
local root, reason = t.root()
if not root then
  warn("No root part")
  return
end

local target = workspace:FindFirstChild("SpawnLocation")
if target then
  t.lookAt(target)
  t.teleport(target)
end

Events And Timing

Use cleaner for connections, callbacks, Instances, and objects with cleanup methods. Use retry, waitUntil, debounce, and throttle to control repeated work.

Cleanup and throttle 
local bin = t.cleaner()

bin:Add(t.on(workspace.ChildAdded, function(child)
  print("Added:", child.Name)
end))

local clicked = t.throttle(1, function()
  print("Only once per second")
end)

bin:Add(function()
  print("cleaned")
end)

bin:Clean()

Effects And HTTP

These helpers depend on the Roblox client or environment. Check the result if your script needs to keep going after a failure.

Tween, notify, HTTP 
t.tween(part, {
  Transparency = 0.35
}, {
  time = 0.4,
  easingStyle = "Quad",
  easingDirection = "Out"
})

t.notify("Prismix", "Script loaded", 3)

local source, reason = t.httpGet("https://example.com")

Remotes And Interactions

Remote and interaction helpers return a value or a clean reason when the target object or function is not supported.

Remote and ClickDetector 
local ReplicatedStorage = t.service("ReplicatedStorage")
local remote = t.get(ReplicatedStorage, "Remotes.Quest")

if remote then
  t.fireRemote(remote, "Start")
end

local button = workspace:FindFirstChild("Button")
local detector = button and button:FindFirstChildOfClass("ClickDetector")

local ok, reason = t.fireClick(detector)
if not ok then
  warn(reason)
end
Only use remote and interaction helpers in places where you are allowed to run that kind of script.

Data And Math

Prismix includes a small set of table and number helpers so beginner scripts can stay readable.

Data helpers 
local settings = t.merge({
  speed = 16,
  jump = 50
}, {
  speed = 24
})

local speed = t.clamp(settings.speed, 0, 100)
local rounded = t.round(1.234, 2)
local hasSpeed = t.contains({ "speed", "jump" }, "speed")

Optional Helpers

Some helpers only register when the current environment supports them. Check with available before using file, clipboard, executor, or runtime helpers.

Check before use 
if t.available("writefile") then
  t.writefile("prismix-log.txt", "ready")
end

local copied, reason = t.copy("Prismix")
if not copied then
  warn(reason)
end

Settings

Set these before loading Prismix.

Settings 
getgenv().PrismixDebug = true
getgenv().PrismixBanner = true

Extend Prismix

Add your own helper with Register, then add a short name with Alias.

Custom helper 
Prismix.Register("spawnPart", function()
  return t.create("Part", {
    Name = "SpawnedByPrismix",
    Anchored = true,
    Parent = workspace
  })
end)

Prismix.Alias("spawn", "spawnPart")
