Start here

Install Prismix first.

Prismix is a small Roblox Luau helper library. It keeps common script tasks short: services, Instances, character parts, child paths, events, tweens, notifications, cleanup, and simple executor checks.

Most helpers return a useful value when they work. If they cannot run, they return nil, reason or false, reason so your script can show a clear warning.
Use Prismix.Tools

All helpers live in one table. Helpers, Functs, and Functions still point to the same table.

Beginner-friendly names

Use short names like service, create, waitForPath, root, on, and cleaner.

Load Prismix 
local Prismix = loadstring(game:HttpGet("https://raw.githubusercontent.com/SairyTheKing/prismix/main/Source.lua"))()
local t = Prismix.Tools

print(Prismix.Info.Version)

What it is for

Prismix removes boring repeat code.

It is not a replacement for learning Luau. It is a helper layer for small scripts where you want cleaner calls and better error messages.

Cleaner Roblox calls

t.service("Players") caches game:GetService. t.create creates Instances and sets Parent last.

Safer script flow

t.try, t.after, t.wait, t.debounce, and t.cleaner help keep small scripts stable.

Optional environment helpers

File, clipboard, executor, and interaction helpers register only when the current environment supports them.

Simple rule

Call a helper, check the result, and warn the reason when it fails.

Return pattern 
local value, reason = t.someHelper()

if not value then
  warn(reason)
  return
end

Beginner example

Create a part without repeating setup.

Pass the class name and a properties table. Prismix applies normal properties first and sets Parent at the end.

1. Load the library

Keep local t = Prismix.Tools near the top so examples stay short.

2. Create the Instance

Use normal Roblox property names inside the table.

3. Check the result

If creation fails, show reason so the problem is easy to find.

Create a part 
local Prismix = loadstring(game:HttpGet("https://raw.githubusercontent.com/SairyTheKing/prismix/main/Source.lua"))()
local t = Prismix.Tools

local part, reason = t.create("Part", {
  Name = "PrismixPart",
  Anchored = true,
  Size = Vector3.new(4, 1, 4),
  Position = Vector3.new(0, 6, 0),
  Parent = workspace
})

if not part then
  warn(reason)
  return
end

t.tween(part, {
  Color = Color3.fromRGB(64, 156, 255)
}, 0.35)

Objects and paths

Find children with short, readable helpers.

Use find when the object may already exist. Use waitFor or waitForPath when Roblox may need a moment to replicate it.

Helper Use it for
t.find(parent, name, recursive)Safe FindFirstChild.
t.waitFor(parent, name, timeout)Safe WaitForChild.
t.get(parent, "A.B.C")Find a dot-separated path now.
t.waitForPath(parent, "A.B.C", timeout)Wait through each part of a path.
The old alias t.path still works, but new scripts should use t.get.
Wait for a remote path 
local ReplicatedStorage = t.service("ReplicatedStorage")

local remote, reason = t.waitForPath(
  ReplicatedStorage,
  "Remotes.Quest",
  10
)

if not remote then
  warn(reason)
  return
end

Player helpers

Get the local player, character, humanoid, and root part.

These helpers accept simple sources. You can pass nothing for the local player, a Player, a character Model, or sometimes a BasePart.

t.player()

Returns Players.LocalPlayer.

t.character(source)

Returns a character model from a player, model, or the local player.

t.root(source)

Returns HumanoidRootPart or another usable BasePart. Alias: rootPart.

t.humanoid(source)

Returns the character's Humanoid.

Move near a target 
local root, reason = t.root()

if not root then
  warn("No root part")
  return
end

local target = workspace:FindFirstChild("SpawnLocation")
if target then
  local ok, moveReason = t.teleport(target)

  if not ok then
    warn(moveReason)
  end
end

Events and cleanup

Connect events, run once, and clean everything later.

Use t.cleaner() when your script creates connections, Instances, or callbacks that need cleanup. This keeps repeated testing and respawns cleaner.

t.on(signal, callback)

Safe wrapper around signal:Connect. Alias: connect.

t.once(signal, callback)

Runs one time, then disconnects.

t.debounce(seconds, callback)

Returns a function that ignores repeat calls until the timer ends.

Cleanup bag 
local bin = t.cleaner()

bin:Add(t.on(workspace.ChildAdded, function(child)
  print("Added:", child.Name)
end))

bin:Add(function()
  print("cleaned")
end)

task.wait(10)
bin:Clean()

Small actions

Tween, notify, copy text, and schedule work.

These helpers depend on the Roblox client or executor environment. Check the return value if your script needs to keep going after a failure.

Helper What it does
t.tween(instance, props, info, play)Creates a Tween and plays it unless play is false.
t.notify(title, text, seconds)Sends a Roblox core notification when supported.
t.copy(text)Copies text if the clipboard API exists.
t.after(seconds, callback, ...)Runs a callback later using task.delay or delay.
t.wait(seconds)Uses task.wait or wait.
Tween and notify 
local part = workspace:FindFirstChild("PrismixPart")

if part then
  t.tween(part, {
    Transparency = 0.35
  }, {
    time = 0.4,
    easingStyle = "Quad",
    easingDirection = "Out"
  })
end

t.notify("Prismix", "Script loaded", 3)

Interaction helpers

Use supported executor actions without hard crashes.

Click, prompt, and touch helpers return false, reason when the function or target object is not supported.

t.fireClick(detector)

Fires a ClickDetector when fireclickdetector exists. Alias: fireClickDetector.

t.firePrompt(prompt)

Fires a ProximityPrompt when supported. Alias: fireProximityPrompt.

t.fireTouch(source, target)

Fires touch interest between your source root and a target BasePart.

Only use interaction helpers in places where you are allowed to run that kind of script.
ClickDetector example 
local button = workspace:FindFirstChild("Button")
local detector = button and button:FindFirstChildOfClass("ClickDetector")

local ok, reason = t.fireClick(detector)

if not ok then
  warn(reason)
end
