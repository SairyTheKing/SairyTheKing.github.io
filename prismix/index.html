Prismix Documentation

Install Prismix first. Prismix is a Roblox helper library for cleaner scripts, safer utility calls, Instance creation, character lookup, path lookup, timing wrappers, and supported executor features.

Version 3.0.1 Single Source.lua Public docs only

Installation

GitHub 
local Prismix = loadstring(game:HttpGet("https://raw.githubusercontent.com/SairyTheKing/prismix/main/Source.lua"))()
local t = Prismix.Tools

What Prismix Is For

Prismix is made to reduce repeated Roblox scripting boilerplate. It wraps common operations with predictable return values, so scripts can fail cleanly instead of crashing without context.

Build Faster Create Instances, assign properties, cache services, and find paths with less repeated code.
Fail Cleaner Most helpers return nil, reason or false, reason when something cannot run.
Stay Flexible Optional executor helpers register only when the current environment supports them.

Main Tools

Group Helpers
Core say, warnmsg, err, try, list, available
Roblox service, create, set, destroy, find, waitFor, path, waitForPath
Character player, character, humanoid, rootPart, teleportTo
Workflow tween, notify, copy, httpGet, after, defer, once, debounce, throttle
Use Prismix.Tools for new scripts. Helpers, Functs, and Functions are aliases for compatibility.

API Reference

Search helpers by name or purpose.

Helper Purpose
service(name)Cached game:GetService.
create(className, properties)Create an Instance and assign Parent last.
set(instance, properties)Assign Instance properties safely.
path(root, "A.B.C")Find a dot-separated child path.
waitForPath(root, "A.B.C", timeout)Wait through a dot-separated child path.
rootPart(source)Resolve local player, Player, Model, or BasePart to a usable part.
tween(instance, goals, info)Create and play a Tween.
debounce(seconds, callback)Return a debounced function wrapper.
throttle(seconds, callback)Return a throttled function wrapper.
copy(text)Copy text when the clipboard API exists.
notify(title, text, duration)Send a Roblox notification when supported.
fireClickDetector(detector)Fire a ClickDetector when supported.
fireTouchInterest(source, target)Fire touch interest when supported.
fireProximityPrompt(prompt)Fire a ProximityPrompt when supported.

Examples

Create A Part

local part, reason = t.create("Part", {
  Name = "PrismixPart",
  Anchored = true,
  Parent = workspace
})

if not part then
  warn(reason)
end

Wait For A Remote Path

local remote, reason = t.waitForPath(game.ReplicatedStorage, "Remotes.Quest", 10)
if not remote then
  warn(reason)
end
