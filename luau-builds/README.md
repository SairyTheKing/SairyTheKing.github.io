# LuauBuilds

Short Roblox Luau tutorials for beginners.

LuauBuilds teaches scripting by small builds:

- Click doors
- Talk GUIs
- Teleport pads
- Screen buttons
- Damage bricks
- Coin pickups
- Light switches
- Speed pads
- Checkpoints
- Tool givers

Open `index.html` to use the site.

No build tools are needed. It is a single static HTML file.
