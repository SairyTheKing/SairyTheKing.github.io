Basics

Script vs LocalScript

  • Script runs on the server.
  • LocalScript runs for one player.
  • Use LocalScripts for player UI.

WaitForChild

Use WaitForChild when a script needs an object that may not be loaded yet.

Debounce

A debounce is a simple cooldown. Use it on Touched events so code does not run too fast.

Test Often

Press Play after each small change. Fix one problem before adding more code.
